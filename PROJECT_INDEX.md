## Summary

Exile Assistant (repo: POEassist) is a single-page, static GitHub Pages app for Path of Exile 2: gear resistance checking (with OCR), a build shopping list, an atlas farm planner with a market radar, two codex tabs, and Today/Exile Hub dashboards. No backend — data is either in the browser's `localStorage` or in static JSON files refreshed by GitHub Actions. (The mock-data Divine Market tab was removed in patch 0.39.)

## Tech style

- One big file: `index.html` (~11290 lines as of 2026-07-14, patch 0.28) — inline `<style>`, inline `<script>`, no build step, no framework.
- Static JSON in `data/` is the only "backend": generated by Node scripts in `scripts/`, committed by a GitHub Actions workflow.
- User-entered data (gear presets, shopping list, farm cards, filters) lives only in browser `localStorage` — never sent anywhere.
- Thai-language comments/strings are intentional throughout `index.html` — keep that style when editing nearby code.
