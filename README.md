# Exile Assistant

**Your personal Path of Exile 2 build, gear, trade, and farming assistant.**

🔗 **Live:** https://paamzuza77.github.io/POEassist/

A single-page, static web app that helps you decide **what to buy next, what to fix, and what to farm now** in PoE2 — without a backend, an account, or your data ever leaving the browser.

## Features

- **🏠 Exile Hub** — league timer + stash-sale countdown (live, from a scheduled snapshot) and a curated resource launcher.
- **📡 Market Radar** — hourly poe.ninja economy snapshot with a transparent **Farm Score** (price trend, value, liquidity, content mapping, risk — never a "drop chance") and a **price lookup** box.
- **🗺️ Farm Planner** — card-based atlas farming strategies with a run tracker (Div/hour).
- **🛡️ Resistance Checker** — resistance/rarity totals with an element radar chart, readiness score, and augment calculator. Fill it from **OCR of a screenshot** or by **pasting item text** (Ctrl+C in game).
- **🛒 Gear Planner** — a build shopping list with a paperdoll gear board, per-slot budgets, and a **paste-item** shortcut that creates rows for you.
- **📖 Content Codex & 🔑 Keyword Codex** — searchable PoE2 boss/content and keyword references.
- **📅 Today** — a daily command dashboard + Kanban todo list.
- **⌘K Command Palette** — Ctrl/⌘+K from anywhere: navigate, run actions, and search.

Three themes (Modern / RawBlock / Trust), light & dark, TH/EN, and a Simple/Advanced density toggle.

## How it works

- **No backend.** Everything you create (gear presets, shopping list, farm cards, todos) lives only in your browser's `localStorage`. Export/Import JSON is available for backup.
- **Market/league data** is generated by GitHub Actions (`scripts/*.mjs`) into static JSON under `data/` on a schedule — the browser never calls poe.ninja / poe2db directly.
- Static hosting on GitHub Pages: one `index.html` + a few `js/`/`css/` helpers, no build step.
